## Getting started

The site is built with [Astro](https://astro.build), TypeScript, and MDX.

**Prerequisites:** Node.js >= 24.13.0 and npm >= 11.0.0

```bash
npm install
npm run dev
```

The development server listens at `http://localhost:4321`.
