# Suporte à Versão

Only the latest version of any given major release line is supported.

Versions that are EOL (end-of-life) _may_ receive updates for critical security vulnerabilities, but the Express team offers no guarantee and does not plan to address or release fixes for any issues found.

| Versão Principal                                                                                                                                           | Versão mínima do Node.js | Início Suporte   | Término Suporte                                            |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------- | ---------------- | ---------------------------------------------------------- |
| [**v5.x**{: .supported }](/{{page.lang}}/5x/api.html){: .ignore-underline} | 18                                       | Setembro de 2024 | **ongoing**{: .supported } |
| [**v4.x**{: .supported }](/{{page.lang}}/4x/api.html){: .ignore-underline} | 0.10.0   | Abril de 2014    | **ongoing**{: .supported } |
| [**v3.x**{: .eol }](/{{page.lang}}/3x/api.html){: .ignore-underline}       | 0.8.0    | Outubro de 2012  | Julho de 2015                                              |
| [**v2.x**{: .eol }](/2x/){: .ignore-underline}                             | 0.4.1    | Março de 2011    | Julho de 2012                                              |
| **v1.x**{: .eol }                                                                                          | 0.2.0    | Dezembro de 2010 | Março de 2011                                              |
| **v0.14.x**{: .eol }                                                                       | 0.1.98   | Dezembro de 2010 | Dezembro de 2010                                           |
