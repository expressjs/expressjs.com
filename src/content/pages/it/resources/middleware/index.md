The Express middleware modules listed here are maintained by the
[Expressjs team](https://github.com/orgs/expressjs/people).

| Modulo Middleware                                        | Descrizione                                                                                                      |
| -------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| [body-parser](/resources/middleware/body-parser)         | Analizza il corpo della richiesta HTTP.                                                                          |
| [compression](/resources/middleware/compression)         | Comprimi le risposte HTTP.                                                                                       |
| [cookie-parser](/resources/middleware/cookie-parser)     | Analizza l'intestazione dei cookie e popola `req.cookies`. Cfr. anche [cookies](https://github.com/jed/cookies). |
| [cookie-session](/resources/middleware/cookie-session)   | Stabilisci sessioni basate sui cookie.                                                                           |
| [cors](/resources/middleware/cors)                       | Abilita la condivisione delle risorse di origine incrociata (CORS) con varie opzioni.                            |
| [errorhandler](/resources/middleware/errorhandler)       | Gestione errori di sviluppo/debug.                                                                               |
| [method-override](/resources/middleware/method-override) | Ignora i metodi HTTP usando l'intestazione.                                                                      |
| [morgan](/resources/middleware/morgan)                   | Registratore di richieste HTTP.                                                                                  |
| [multer](/resources/middleware/multer)                   | Gestire i dati multi-parte.                                                                                      |
| [response-time](/resources/middleware/response-time)     | Registra il tempo di risposta HTTP.                                                                              |
| [serve-favicon](/resources/middleware/serve-favicon)     | Servire una favicon.                                                                                             |
| [serve-index](/resources/middleware/serve-index)         | Serve l'elenco delle directory per un percorso specificato.                                                      |
| [serve-static](/resources/middleware/serve-static)       | Servire file statici.                                                                                            |
| [session](/resources/middleware/session)                 | Stabilisci sessioni basate su server (solo sviluppo).                                                            |
| [timeout](/resources/middleware/timeout)                 | Imposta un timeout elaborazione richiesta perioHTTP.                                                             |
| [vhost](/resources/middleware/vhost)                     | Crea domini virtuali.                                                                                            |
