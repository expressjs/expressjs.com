title: Express-Middleware
description: Entdecken Sie eine Liste der Express.js Middleware-Module, die vom Express-Team und der Community verwaltet werden, einschließlich eingebauter Middleware und beliebter Module von Drittanbietern.
---

Die hier aufgelisteten Express-Middleware-Module werden vom
[Expressjs Team](https://github.com/orgs/expressjs/people) betreut.

| Middleware-Modul                                            | Beschreibung                                                                                                 |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| [body-parser](/en/resources/middleware/body-parser)         | HTTP-Request-Körper analysieren.                                                                             |
| [compression](/en/resources/middleware/compression)         | HTTP Antworten komprimieren.                                                                                 |
| [cookie-parser](/en/resources/middleware/cookie-parser)     | Parsen Sie Cookie-Header und füllen Sie `req.cookies`. Siehe auch [cookies](https://github.com/jed/cookies). |
| [cookie-session](/en/resources/middleware/cookie-session)   | Cookie-basierte Sitzungen einrichten.                                                                        |
| [cors](/en/resources/middleware/cors)                       | Quellenübergreifende Ressourcenfreigabe (CORS) mit verschiedenen Optionen aktivieren.                        |
| [errorhandler](/en/resources/middleware/errorhandler)       | Entwicklung Fehlerbehandlung/Debugging.                                                                      |
| [method-override](/en/resources/middleware/method-override) | HTTP-Methoden mit Header überschreiben.                                                                      |
| [morgan](/en/resources/middleware/morgan)                   | HTTP Request Logger.                                                                                         |
| [multer](/en/resources/middleware/multer)                   | Mehrteilige Formulardaten behandeln.                                                                         |
| [response-time](/en/resources/middleware/response-time)     | HTTP-Antwortzeit aufzeichnen.                                                                                |
| [serve-favicon](/en/resources/middleware/serve-favicon)     | Servieren Sie ein Favicon.                                                                                   |
| [serve-index](/en/resources/middleware/serve-index)         | Verzeichnisliste für einen angegebenen Pfad Servieren.                                                       |
| [serve-static](/en/resources/middleware/serve-static)       | Statische Dateien Servieren.                                                                                 |
| [session](/en/resources/middleware/session)                 | Server-basierte Sitzungen einrichten (nur Entwicklung).                                                      |
| [timeout](/en/resources/middleware/timeout)                 | Setze eine Timeout PerioHTTP Request-Verarbeitung.                                                           |
| [vhost](/en/resources/middleware/vhost)                     | Virtuelle Domains anlegen.                                                                                   |

## Zusätzliche Middleware-Module

Dies sind einige weitere beliebte Middleware-Module.

<Alert type="warning">
  Diese Informationen beziehen sich auf Sites, Produkte oder Module, die nicht vom
  Expressjs Team betreut werden. Die Auflistung hier stellt keine Billigung oder Empfehlung des Projektteams von
  Expressjs dar.

| Middleware-Modul                                    | Beschreibung                                                                                                                                        |
| --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |

| [passport](https://github.com/jaredhanson/passport) | Authentifizierung mit "Strategien" wie OAuth, OpenID und vielen anderen. Siehe [passportjs.org](https://passportjs.org/) für weitere Informationen. |
