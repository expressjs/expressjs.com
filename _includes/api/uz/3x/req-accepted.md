<h3 id='req.accepted'>req.accepted</h3>

Return an array of Accepted media types ordered from highest quality to lowest.

~~~js
[ { value: 'application/json',
    quality: 1,
    type: 'application',
    subtype: 'json' },
   { value: 'text/html',
     quality: 0.5,
     type: 'text',
     subtype: 'html' } ]
~~~
