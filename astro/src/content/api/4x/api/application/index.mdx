## Application

The `app` object conventionally denotes the Express application.
Create it by calling the top-level `express()` function exported by the Express module:

```js
var express = require('express');
var app = express();

app.get('/', function (req, res) {
  res.send('hello world');
});

app.listen(3000);
```

The `app` object has methods for

- Routing HTTP requests; see for example, [app.METHOD](/api/application/app-method) and [app.param](/api/application/app-param).
- Configuring middleware; see [app.route](/api/application/app-route).
- Rendering HTML views; see [app.render](/api/application/app-render).
- Registering a template engine; see [app.engine](/api/application/app-engine).

It also has settings (properties) that affect how the application behaves;
for more information, see [Application settings](/api/application/app-set#app.settings.table).

<Alert type="info">

The Express application object can be referred from the [request object](/api/request) and the [response object](/api/response) as `req.app`, and `res.app`, respectively.

</Alert>
